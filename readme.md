# Fan Sport — Интернет-магазин спортивных товаров

Современный и легковесный интернет-магазин спортивных товаров, написанный на Django и Tailwind CSS. Поддерживает регистрацию, авторизацию пользователей, динамический каталог товаров с поиском и фильтрацией по категориям из базы данных, интерактивную AJAX-корзину, личный кабинет со списком заказов и встроенный симулятор безопасных онлайн-платежей ЮKassa.

---

## Требования к системе
Перед запуском убедитесь, что у вас установлены:
– Python (версии 3.10 или новее)
– PostgreSQL (версии 13 или новее)

---

## Быстрый старт для Windows (Инструкция по запуску)

Выполните эти шаги по порядку в терминале вашего редактора кода (или в PowerShell / Командной строке).

### 1. Подготовка виртуального окружения
Создайте изолированное окружение и активируйте его.

**Для Windows PowerShell:**
```powershell
python -m venv .venv
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
.venv\Scripts\activate
```

**Для классической Командной строки (CMD) Windows:**
```cmd
python -m venv .venv
.venv\Scripts\activate.bat
```

### 2. Установка библиотек
Установите Django и драйвер базы данных внутри активированного виртуального окружения:
```cmd
python -m pip install -r requirements.txt
```

### 3. Настройка базы данных
1. Откройте клиент PostgreSQL (pgAdmin или консоль psql) и создайте базу данных:
   ```sql
   CREATE DATABASE fansport_db;
   ```
2. Откройте файл `DjangoProject/settings.py` и в блоке `DATABASES` пропишите ваш реальный пароль от PostgreSQL (вместо `ВАШ_ПАРОЛЬ`):
   ```python
   DATABASES = {
       'default': {
           'ENGINE': 'django.db.backends.postgresql',
           'NAME': 'fansport_db',
           'USER': 'postgres',
           'PASSWORD': 'ВАШ_ПАРОЛЬ',
           'HOST': '127.0.0.1',
           'PORT': '5432',
       }
   }
   ```

### 4. Применение таблиц и наполнение товарами
Создайте структуру таблиц в базе и автоматически залейте в каталог 20 готовых спортивных товаров по категориям:
```cmd
# Применяем миграции
python manage.py makemigrations library
python manage.py makemigrations users
python manage.py migrate

# Наполняем базу 20 товарами (из кода)
python manage.py seed_products
```

### 5. Создание администратора
Создайте профиль администратора, чтобы заходить в панель управления:
```cmd
python manage.py createsuperuser
```
*(Консоль предложит вам ввести имя пользователя, почту и дважды пароль).*

### 6. Запуск сервера
Запустите сервер разработки:
```cmd
python manage.py runserver
```

Теперь откройте браузер и перейдите по адресу:
– Главная страница (Каталог товаров): http://127.0.0.1:8000/
– Личный кабинет пользователя: http://127.0.0.1:8000/accounts/profile/
– Панель администратора (Django Admin): http://127.0.0.1:8000/admin/
